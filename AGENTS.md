# AGENTS.md — NudgeMug

This repo is a **fun-mode mini project**.

## Prime directive

Ship a cute, usable thing fast. This is a tiny side project, not another cathedral.

## Constraints

- Deadline: **Tue 2026-06-02 14:57 GMT+2**.
- Optimize for joy, clarity, and a working demo.
- CLI first. GUI later.
- Keep files small and obvious.
- Prefer boring code over clever code.
- Do not turn this into a platform, daemon framework, plugin system, cloud service, or compliance exercise.

## Coding style

- Small Go packages.
- No dependency unless it pays rent immediately.
- Friendly CLI output.
- Good README > perfect internals.
- If a choice takes more than 10 minutes, pick the simpler one and move.

## Product vibe

Tiny app. Polite mouse nudge. Slightly funny. Public GitHub repo energy.

Possible tagline:

> Keep your laptop politely alive.

## Done means

A normal person can clone it, run it, understand it, and smile.
